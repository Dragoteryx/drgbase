## Available Tools

### drgbase_tool_info
View detailed information about an NPC including:
- Health and status
- Current AI state
- Movement information
- Animation state
- Possession status
- Relationships

### drgbase_tool_damage
Apply precise damage to NPCs for testing.

### drgbase_tool_faction
Change an NPC's faction on the fly.

### drgbase_tool_relationship_simple
Set basic relationships between entities.

### drgbase_tool_disableai
Toggle AI on/off.

### drgbase_tool_omniscient
Enable omniscient mode (NPC sees all enemies).

### drgbase_tool_notarget
Make entity invisible to AI.

### drgbase_tool_godmode
Toggle invincibility.

### drgbase_tool_mover
Move entities precisely.

### drgbase_tool_scale
Scale entity size.

## Using Tools

1. Select tool from toolgun menu
2. Look at target NPC
3. Left/right click for actions
4. Check tool's specific documentation for details
