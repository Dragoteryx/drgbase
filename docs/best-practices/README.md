## Quick Tips

### Performance
- Use appropriate AI radius
- Minimize network messages
- Cache frequently used values
- Disable precaching if not needed

### Code Quality
- Comment your code
- Use meaningful variable names
- Follow Lua conventions
- Keep functions focused

### Testing
- Test with multiple NPCs
- Test on multiplayer servers
- Use debug tools
- Monitor console for errors

### Compatibility
- Don't modify DrGBase core files
- Use proper inheritance
- Check for nil values
- Handle edge cases
